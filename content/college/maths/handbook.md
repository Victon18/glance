# Differentiation


 1. $\Large \frac{d}{dx}(x^{n})= nx^{n-1}$
2. $\Large \frac{d}{dx}(a^{x})= a^{x}log_{e}a$
3.  $\Large \frac{d}{dx}(e^{x})= e^{x}$
4.  $\Large \frac{d}{dx}(log_{e}x)= \frac {1}{x}$
5.  $\Large \frac{d}{dx}(log_{10}x)= \frac {1}{x}log_{10}e$

## Trignometry
### basic

1. $\LARGE \frac{d\sin x}{dx}=\cos x$
2. $\LARGE \frac{d\cos x}{dx}=-\sin x$
3. $\LARGE \frac{d\tan x}{dx}=\sec^{2} x$
4. $\LARGE \frac{d\cot x}{dx}=-\csc^{2} x$
5. $\LARGE \frac{d\sec x}{dx}=\sec x\tan x$
6. $\LARGE \frac{d\csc x}{dx}=-\csc x\cot x$
### inverse

1.  $\LARGE \Large \frac{d}{dx}(\sin^{-1}x)= \frac{1}{\sqrt{1-x^{2}}}$
2.   $\LARGE \Large \frac{d}{dx}(\cos^{-1}x)= \frac{-1}{\sqrt{1-x^{2}}}$
3.   $\LARGE \Large \frac{d}{dx}(\tan^{-1}x)= \frac{1}{1+x^{2}}$
4.   $\LARGE \Large \frac{d}{dx}(\sec^{-1}x)= \frac{1}{x\sqrt{x^{2}-1}}$
5.   $\LARGE \Large \frac{d}{dx}(\cot^{-1}x)= \frac{-1}{1+x^{2}}$
6.   $\LARGE \frac{d}{dx}(\csc^{-1}x)= \frac{-1}{x\sqrt{x^{2}-1}}$
### hyperbolic

1. $\Large \frac{d}{dx}(\sin h x)= \frac{e^{x}-e^{-x}}{2}$
2. $\Large \frac{d}{dx}(\cos h x)= \frac{e^{x}+e^{-x}}{2}$
3. $\Large \frac{d}{dx}(\tan hx)= \frac{e^{x}-e^{-x}}{e^{x}+e^{-x}}$

## 3x and 2x

1. $\LARGE \sin 3x = 3\sin x-4\sin^{3}x$
2. $\LARGE \cos 3x = 4\cos^{3}x-3\cos x$
3. $\LARGE \tan3x = \frac{3\tan x-\tan^{3}x}{1-3\tan^{2}x}$
4. $\LARGE sin2x = 2sinx\cdot cosx$
5. $\LARGE tan2x = \frac{2tanx}{1-tan^{2}x}$
6. $\LARGE cos2x = 2cos^{2}x-1=1-2sin^{2}x=cos^{2}x-sin^{2}x=\frac{1-tan^{2}x}{1+tan^{2}x}$

# Integration

 1. $\Large\int1\;dx = x+C$
2. $\Large\int\;x^{n}\;dx=\frac{x^{(n+1)}}{(n+1)}+C$
3.  $\Large\int\;\frac{1}{x}\;dx=\log|x|+C$
4. $\Large\int\;e^{x}\;dx=e^{x}+C$
5. $\Large\int\;a^{x}\;dx=\frac{a^{x}}{\log a}+C$
6. $\Large\int\;e^{x}[f(x)+\frac{d}{dx}f(x)]\;dx=e^{x}f(x)+C$
## Trignometry
### basic

 1. $\Large\int\cos x\;dx = \sin x+C$
 2. $\Large\int\sin x\;dx = -\cos x+C$
 3. $\Large\int\sec^2{} x\;dx = \tan x+C$ 
 4. $\Large\int\csc^{2} x\;dx = -\cot x+C$
 5. $\Large\int\sec x\tan x\;dx = \sec x+C$
 6. $\Large\int\csc x\cot x\;dx = -\csc x+C$
 7. $\Large\int\tan x\;dx = \log|\sec x|+C$
 8. $\Large\int\cot x\;dx = \log|\sin x|+C$
 9. $\Large\int\sec x\;dx = \log|\sec x+\tan x|+C=\log\tan(\frac{\pi}{4}+\frac{x}{2})+C$
 10. $\Large\int\csc x\;dx = \log|\csc x-\cot x|+C=\log\tan\frac{x}{2}+C$
### inverse

 1. $\Large\int\frac{1}{\sqrt{a^{2}-x^{2}}}\;dx=\sin^{-1}\frac{x}{a}+C$
 2. $\Large\int-\frac{1}{\sqrt{a^{2}-x^{2}}}\;dx=\cos^{-1}\frac{x}{a}+C$
 3. $\Large\int\frac{1}{a^{2}+x^{2}}\;dx=\frac{1}{a}\tan^{-1}\frac{x}{a}+C$
 4. $\Large\int-\frac{1}{a^{2}+x^{2}}\;dx=\frac{1}{a}\cot^{-1}\frac{x}{a}+C$
 5. $\Large\int\frac{1}{x\sqrt{x^{2}-a^{2}}}\;dx=\frac{1}{a}\sec^{-1}\frac{x}{a}+C$
 6. $\Large\int-\frac{1}{x\sqrt{x^{2}-a^{2}}}\;dx=\frac{1}{a}\csc^{-1}\frac{x}{a}+C$
### others

 1. $\Large\int\frac{1}{a^{2}-x^{2}}\;dx=\frac{1}{2a}\log(\frac{a+x}{a-x})+C$
 2. $\Large\int\frac{1}{x^{2}-a^{2}}\;dx=\frac{1}{2a}\log(\frac{x-a}{x+a})+C$
### special

1. $\Large\int\frac{1}{x^{2}-a^{2}}\;dx=\frac{1}{2}a\log|(x-a)(x+a)|+C$
2. $\Large\int\frac{1}{a^{2}-x^{2}}\;dx=\frac{1}{2}a\log|(a+x)(a-x)|+C$
3. $\Large\int\frac{1}{x^{2}+a^{2}}\;dx=\frac{1}{2}a\tan^{-1}\frac{x}{a}+C$
4. $\Large\int\frac{1}{\sqrt{x^{2}-a^{2}}}\;dx=\log|x+\sqrt{x^{2}-a^{2}}|+C$
5. $\Large\int\frac{1}{\sqrt{x^{2}+a^{2}}}\;dx=\log|x+\sqrt{x^{2}+a^{2}}|+C$
6. $\Large\int\frac{1}{\sqrt{a^{2}-x^{2}}}\;dx=\sin^{-1}\frac{x}{a}+C$
7. $\Large\int\sqrt{x^{2}-a^{2}}\;dx=\frac{x}{2}\sqrt{x^{2}-a^{2}}-\frac{a^{2}}{2}\log|x+\sqrt{x^{2}-a^{2}}|+C$
8. $\Large\int\sqrt{x^{2}+a^{2}}\;dx=\frac{x}{2}\sqrt{x^{2}+a^{2}}+\frac{a^{2}}{2}\log|x+\sqrt{x^{2}+a^{2}}|+C$
9. $\Large\int\sqrt{a^{2}-x^{2}}\;dx=\frac{x}{2}\sqrt{a^{2}-x^{2}}-\frac{a^{2}}{2}\sin^{-1}\frac{x}{a}+C$

### parabolic 

1. $\Large\int\sec h^{2}\;x\;dx=\tan h\;x+C$ 
2. $\Large\int\csc h^{2}\;x\;dx=-\cot h\;x+C$ 
3. $\Large\int\sin h\;x\;dx=\cos h\;x+C$ 
4. $\Large\int\cos h\;x\;dx=\sin h\;x+C$ 
5. $\Large\int\frac{1}{\sqrt{x^{2}-a^{2}}}\;dx=\cos h^{-1}\frac{x}{a}+C$
6. $\Large\int\frac{1}{\sqrt{a^{2}+x^{2}}}\;dx=\sin h^{-1}\frac{x}{a}+C$

#### definite

1. $\Large\int^{b}_{a}f(x)\;dx=\int^{b}_{a}f(y)\;dy+C$
2. $\Large\int^{b}_{a}f(x)\;dx=-\int^{a}_{b}f(x)\;dx+C$
3. $\Large\int^{a}_{0}f(x)\;dx=\int^{a}_{0}f(a-x)\;dx+C$
