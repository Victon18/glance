# Escape characters
```python
print('Man\'s new best friend')
```
---
```
`\n` -> newline
`\t` -> tab
`\\` -> single backslash
`\'` -> single quote
'\"` -> double quote
`\r` -> carraige return
`\b` -> backspace
`\f` -> form feed
`\v` -> vertical tab
```

# slicing and indexing
```python
x = "Hello my friend great to see you"
print(x[10])
print(x[0:10])
print(x[1:12:2])
print(x[:9])
print(x[3:])
print(x[::])
print(x[::-1])
```
